# Card Classifier

Buzzfeed style app that outputs the card color in playing card universes (e.g. Magic The Gathering, Pokemon) from an
input images. 

# Procedure

- `cd card-classifier`
- `pip install -e .`
- [Download data]

# Data

### Magic The Gathering:

- Download raw data locally with `download_magic`
